# MulBan-CTNet Extended Dataset

**11,197 balanced text-image pairs** for multimodal Bangla fake news detection.

### Files
- `*.csv` → Metadata, labels, text, and image paths (3 files)
- `thesis-images.zip` → All thumbnail images

### Download
**Recommended (faster & better organized)**:  
[Hugging Face Dataset](https://huggingface.co/datasets/thesiswork67/thesis-work-dataset)  
[Hugging Face Images](https://huggingface.co/datasets/thesiswork67/thesis-image-dataset)
